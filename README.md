# Interactive Data Visualizer

An interactive data visualization and preprocessing web application built with **Streamlit**, **Pandas**, and **Plotly**. Upload a CSV or Excel file, clean and filter your data, generate interactive charts, discover insights, and export the processed dataset—all in one place.

## ✨ Features

* Upload CSV and Excel datasets
* View dataset summary, statistics, and column information
* Handle missing values and remove duplicate records
* Filter data using numeric ranges and categorical values
* Automatically detect and encode binary (Yes/No, True/False) columns
* Create interactive charts (Line, Bar, Scatter, Histogram, Pie, Box, Violin, Area, Scatter Matrix)
* Generate grouped aggregations and correlation heatmaps
* Get automatic insights on missing values, skewness, and feature correlations
* Export the cleaned dataset as a CSV

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly Express

## 🚀 Getting Started

```bash
git clone https://github.com/<your-username>/interactive-data-visualizer.git
cd interactive-data-visualizer

pip install -r requirements.txt

streamlit run app.py
```
